# LLMGatewayKit

Swift client SDK for [llm-gateway-back](https://github.com/snana7mi/llm-gateway-back), plus shared account, subscription, and sync infrastructure used by SnapKei and ConchTalk.

## Install

```swift
.package(url: "https://github.com/snana7mi/LLMGatewayKit", from: "0.1.0")
```

## Status

0.1.x - public API may change before 1.0.
